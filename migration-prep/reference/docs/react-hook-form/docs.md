# API

React Hook Form's API overview

<div class="ApiGallery_root__Qke3P" role="main">

- <div>

  ### `</>`useForm

  A powerful custom hook to validate your form with minimal re-renders.

  <a href="/docs/useform" aria-label="read more about useForm">Read More ▸</a>

  </div>

- <div>

  ### `</>`useController

  For Controlled components: interface with the useForm methods and isolate its re-render.

  <a href="/docs/usecontroller" aria-label="read more about useController">Read More ▸</a>

  </div>

- <div>

  ### `</>`useFormContext

  Access your useForm methods and properties from nested components. Great for building larger forms and shared components!

  <a href="/docs/useformcontext" aria-label="read more about useformcontext">Read More ▸</a>

  </div>

- <div>

  ### `</>`useWatch

  Subscribe to individual form input changes without impacting the root component's render.

  <a href="/docs/usewatch" aria-label="read more about usewatch">Read More ▸</a>

  </div>

- <div>

  ### `</>`useFormState

  Subscribe to individual form state updates and isolating re-renders at the hook level.

  <a href="/docs/useformstate" aria-label="read more about useformstate">Read More ▸</a>

  </div>

- <div>

  ### `</>`useFieldArray

  Manage dynamically generated fields on the fly, shuffle, remove and append fields. Ideal for complex CRUD data entry scenarios.

  <a href="/docs/usefieldarray" aria-label="read more about usefieldarray">Read More ▸</a>

  </div>

- <div>

  ### `</>`useLens

  Type-safe lenses for building reusable and shareable form components with precise focus on form fields.

  <a href="/docs/uselens" aria-label="read more about useLens">Read More ▸</a>

  </div>

- <div>

  ### `</>`createFormControl

  Create form control object and subscribe form state outside of React component, access methods without context.

  <a href="/docs/createFormControl" aria-label="read more about usefieldarray">Read More ▸</a>

  </div>
